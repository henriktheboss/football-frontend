# Football Frontend

Enkel React + Next.js frontend for å vise og opprette fotballspillere ved bruk av football-backend.

## Teknologistack

- Next.js
- React
- TypeScript
- Fetch API
- Ingen eksterne UI-rammeverk

## Hvordan kjøre

### Krav
- Node.js 18+
- npm

### Steg

Installer avhengigheter:

```bash
npm install
```

Start app: 

```bash
npm run dev
```
